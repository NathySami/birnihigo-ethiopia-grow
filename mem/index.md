# Project Memory

## Core
Brand: Birnihigo Integrated Farms. Primary Café Noir #4F3C1C, Secondary Deep Saffron #FEA42A. Accents White Chocolate #EFE7DC, Dandelion #FFD275, Chinese Bronze #CD8C24. Typeface Parkinsans (display + body). Color proportions: 70% primary / 20% secondary / 10% accent. Tagline: "Feeding the Future, Naturally." Source: Brand Guidelines pp.11-15.
Voice: bold, decisive, strategic — "system not farm", "platform not production", "infrastructure not facilities". Position as a national-scale, climate-resilient protein platform. Always mention Afar resilience, Moringa, BSF, solar/biogas, UN SDGs, structured contract farming integrating thousands of smallholders. Hatchery/parent-stock icons must be egg-based.
